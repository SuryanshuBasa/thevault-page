Capabilities

What makes it
trustworthy

Offline-First Capture
Jots save to local storage before any network call, so Today works with zero signal — poolside, gym basement, a flight. It syncs the moment it reaches home.
IndexedDB · sync-on-reconnect
Never-Lose-A-Note Sync
The phone is the single author of each day and overwrites idempotently. If local storage is ever cleared, it restores the day from the server on next open — a wipe can't lose your words.
Model A · restore-on-empty
You-First Chat
Every session loads your charter, living memory, and recent daily logs before you say a word — then retrieves across your knowledge wikis. The same depth as desktop, in your pocket.
charter + memory + RAG
Writes Only On Your Word
Memory and plan edits are never silent — the agent drafts the exact file and asks first. Capture is the only no-confirm write, because it's your own raw note. Surface and recommend, never decide.
draft → confirm → write
Private Remote Access
Reachable from anywhere over a Tailscale tunnel only your own devices share. No password needed because the network itself is the lock — never a public door to the open internet.
Tailscale · one URL
Updates Without Reinstalling
New code lands on a normal reopen — the app re-checks its service worker and reloads into the latest version. No deleting the home-screen app, which is exactly what would wipe your local notes.
PWA · auto-update
Built to Study In
A long answer isn't a chat log — it opens in a full-screen serif reader, with rendered math, code, and tappable source pages. Highlight any passage to ask a scoped follow-up or send the snippet straight to Today. Designed to live in, not just query.
reading mode · select-to-act
